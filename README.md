# Intern Performance Prediction & Analytics System

## Overview
This system uses a trained Machine Learning Regression model (Gradient Boosting Regressor) to predict intern performance based on three core metrics: task completion, consistency, and engagement.

## Sprint Architecture
- **Sprint 1**: Data simulation (`data_generator.py`) and processing (`data_preprocessing.py`).
- **Sprint 2**: Baseline model training and initial comparison (`model_training.py`).
- **Sprint 3**: Hyperparameter tuning, scaling, and serialization (`model_optimization.py`). Evaluation report generated as `evaluation_report.md`.
- **Sprint 4**: API Deployment (`app.py`, `requirements.txt`, `README.md`).

## Setup & Deployment
1. **Install Requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run API Server Locally:**
   ```bash
   uvicorn app:app --reload
   ```
   The endpoint will be available at `http://127.0.0.1:8000/`.

## Prediction API Reference
**Endpoint:** `POST /predict`

**Body Structure:**
```json
{
  "task_completion_rate": 0.85,
  "consistency_score": 0.90,
  "engagement_metric": 0.70
}
```

**Response Structure:**
```json
{
  "predicted_performance_score": 83.45
}
```

View the interactive Swagger UI documentation generated by FastAPI at `http://127.0.0.1:8000/docs`.
